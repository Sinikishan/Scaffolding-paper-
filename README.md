Agent-Orchestrated Decision Support for Logical Framework Development
Overview

This repository contains the conceptual implementation resources, pseudocode, evaluation structure, and supporting artefacts associated with the research paper:

“Scaffolding Program Design with Agent-Orchestrated Decision Support: A Design Science Approach to Logical Framework Development.”

The project proposes a structured, human-in-the-loop decision-support framework that scaffolds Logical Framework (LogFrame) development using an agent-orchestrated architecture grounded in Design Science Research (DSR).

Objectives

The framework aims to:

Support coherent and evaluable program design
Reduce dependence on expert-led facilitation
Improve consistency in Logical Framework development
Enable structured reasoning through modular AI agents
Preserve transparency, traceability, and human oversight
Core Components

The proposed architecture includes:

Component	Function
Planner Agent	Sequences the design workflow
Context Agent	Maintains contextual constraints and scope
Guidance Agent	Provides structured prompts and scaffolding
Validation Agent	Performs consistency and alignment checks
Memory Agent	Stores design rationale and revisions
Human-in-the-Loop Interface	Preserves user control and approval
Workflow Stages

The framework supports the following stages:

Problem Framing

Indicator Alignment

Evaluation Focus

The framework is evaluated based on:

Efficiency of program design
Logical completeness
Coherence and alignment
Stakeholder mapping quality
User confidence and capability
Research Methodology

The study follows a Design Science Research (DSR) methodology involving:

Problem identification
Artefact design
Framework development
Demonstration
Structured evaluation
Data Availability

The datasets, evaluation artefacts, implementation details, and supporting materials associated with this study will be made available following publication of the article through an appropriate public repository or upon reasonable academic request.

Ethical Considerations

The study involved voluntary participation from adult program practitioners for evaluation purposes. No sensitive personal or clinical data were collected. All participant responses were anonymised and used strictly for academic research purposes.
